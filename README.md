#### The Hard Way

```sh
virtualenv -p python3 VENV
. ./VENV/bin/activate
pip install -r requirements.txt
cp sample_config.py config.py
--- EDIT config.py values appropriately ---
python bot.py
```
## Bot Commands

go -upload to gofiles

clearffmpegmedia - clear ffmpeg media
downloadmedia - download media
c2a -convert to audio
c2f-convert to file
c2v-convert to video
set_thumb-set custom thumbnail
sshot -generate screenshot
server - check server details 
tsh - upload to transfer sh
me- my info
ren- rename files
del_thumb - clear custom thumbnail
